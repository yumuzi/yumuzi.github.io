  • 安装完毕，可以通过命令查看node.js版本

Ubuntu 安装和配置 Git

  • 安装Git
  • 配置 ssh-key
  • Git初始设置
  • 提交秘钥到github

会显示 “idrsa idras.pub“ 两个文件，前一个是私钥，后一个是公钥。将公钥粘贴到你的github账号中的 SSH Public Keys 处即可。
